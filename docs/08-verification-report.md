- PDF 原始资料：`docs/09小明课堂 历年真题.pdf` 与 `docs/10小明课堂 《历年真题答案》.pdf`。

## 2. 当前已验证玩法

- 首屏为开局台，显示“今日推荐”“本局目标”“开始一局”。
- 支持三种 Roguelite 模式：探索局、净化局、冲刺局。
- 支持三种流派：稳修、突击、复盘。
- 有活跃心魔时，推荐逻辑会优先给出净化局和复盘流派。
- 推荐局能创建 5 题题阵。
- 局内显示本局目标、遭遇序号、题干、选项和破招式。
- 答错会生成或强化心魔，并记录错因。
- 战报会汇总本局表现，并给出下一局建议。
- 独立剧情主入口已退出主流程，剧情改为开局、遭遇和结算中的事件化短文案。
- 主界面不再使用旧“正式主线”“章节提示”“完成剧情”等玩家可见文案。
