# 小明书院：秘卷巡游 — UI 设计规范（设计版）

> 文档定位：纯文字界面的设计蓝图。用 CSS + Unicode 字符创造丰富的信息可视化，零图片但绝不简陋。

---

## 1. 设计原则

1. **信息密度高**：4680 源题位、4374 当前可玩题、7 主题、6 题型、知识图谱——大量信息需要清晰展示
2. **Unicode 即图形**：用 ▓░█▒╱╲┌┐└┘ 等字符构建图表、进度条、树形图
3. **色彩即状态**：不同状态用不同色彩，一眼识别
4. **层级即焦点**：当前最重要的信息最大，次要的折叠或缩小
5. **反馈即时**：每个操作有即时视觉反馈，延迟 < 200ms

---

## 2. 色彩系统

```css
:root {
  color-scheme: light;
  --bg-page: #f7f1e6;          /* 页面底色：暖米色 */
  --bg-panel: #fffaf2;         /* 面板底色：暖白色 */
  --bg-locked: #f0e8dc;       /* 锁定底色：浅灰米 */
  --text-main: #211b18;        /* 主文字：深墨棕 */
  --text-muted: #70665d;       /* 辅助文字：灰棕 */
  --text-line: #ddd1c2;        /* 边框：浅暖灰 */
  --text-strong: #b9aa98;      /* 按钮边框：中暖灰 */
  --accent: #2d6f62;           /* 青绿：强调/正确/选中/解锁 */
  --accent-light: #edf8f4;     /* 浅青绿：选中背景 */
  --danger: #9d3c45;            /* 朱红：错误/心魔/危险 */
  --danger-light: #fdf2f3;     /* 浅朱红：错误背景 */
  --gold: #c4953a;             /* 金色：成就/掌握/特殊 */
  --gold-light: #fdf6e8;       /* 浅金色：成就背景 */
  --warning: #d4a017;          /* 琥珀：警告/注意 */
  --shadow: 0 12px 32px rgba(55, 43, 31, 0.08);
  --radius: 8px;
}
```

**状态色映射：**

| 状态 | 文字 | 边框 | 背景 | 标记 |
|---|---|---|---|---|
| 默认 | main | strong | panel | — |
| 选中 | accent | accent | accent-light | ✓ |
| 正确 | accent | accent | — | ✓ 绿色 |
| 错误 | danger | danger | danger-light | ✗ 红色 |
| 锁定 | muted | line | locked | 🔒 |
| 心魔 | danger | danger | — | 👹 |
| 掌握 | gold | gold | gold-light | ⭐ |
| 解锁 | accent | accent | — | 🔓 |
| 警告 | warning | warning | — | ⚠ |

**判断题：**
```

（部分关键词被迷雾遮挡：平__接受义务教育的条__）

A. 正确
B. 错误

[释放破招] [回练功] [回地图]

```

**多选题（概念迷宫）：**
```
题干：下列属于正强化的是？
（选项变形：措辞相似）

A. 给予学生表扬（正强化）
B. 撤销学生的惩罚（负强化 ← 变形！）
C. 让学生参加喜欢的活动（正强化）
D. 停止批评学生（负强化 ← 变形！）

提示（观照）：正强化 = 给予想要的；负强化 = 撤销不想要的。两者都增加行为概率，但方式不同。

[释放破招] [回练功]
```

**案例分析（时间沙漏）：**
```
⏳ 剩余时间：45秒

题干：某班学生小明在课堂上突然呕吐...
[材料：200字...]

问题：教师首先应该？

A. 安抚小明情绪
B. 保护现场安全
C. 通知校医
D. 清理呕吐物

[释放破招] [时间紧迫！]
```

**论述题（策略选择）：**
```
题干：请简述教师职业道德的核心内容。

这不是一道选择题。请选择你的答题策略：

A. 先列要点，再展开（观照思路）
   → 提示：核心是爱岗敬业、关爱学生、教书育人、为人师表、终身学习
   → 收益较低，但稳妥

B. 直接展开论述（强攻风格）
   → 无提示，全靠记忆
   → 收益高，但风险大

C. 稳妥分析，不求全面（稳破风格）
   → 只答最有把握的要点
   → 保底得分，但上限低

[选择策略]
```

## 5. 响应式规则

| 元素 | 桌面（≥761px） | 移动（≤760px） |
|---|---|---|

## 6. 动画与反馈（纯CSS）

| 效果 | 实现方式 | 时长 |
|---|---|---|
| 节点点亮 | 边框从灰→绿，文字颜色渐变 | 300ms |
| 心魔诞生 | 节点边框闪烁红→暗红 | 500ms |
| 答题正确 | 边框绿色 + 轻微缩放 | 200ms |
| 答题错误 | 边框红色 + 微震动 | 300ms |
| 进度条增长 | width 过渡 | 500ms |
| 流派选中 | 边框发光 + 背景渐变 | 200ms |
| 封印解锁 | 金色边框 + 缩放脉冲 | 1s |

**减弱动画：** 支持 `prefers-reduced-motion`，关闭时所有效果变为即时切换。
