# 小明书院：秘卷巡游 — 美术素材规范（设计版）

> 文档定位：零图片资源声明。纯文字不代表简陋——用 CSS + Unicode 字符构建丰富的视觉层次。

---

## 1. 设计原则

1. **零图片**：不生成、不引用、不依赖任何图片文件
2. **Unicode 即图形**：▓░█▒┌┐└┘│╱╲✓✗⭐🔒👹⏳ 等字符构建图表、图标、装饰
3. **CSS 即美术**：边框、阴影、渐变、动画全部用 CSS 实现
4. **排版即氛围**：宋体/衬线字体营造书卷感，等宽字体用于数据展示
5. **色彩即情绪**：暖色背景 = 书卷，青绿 = 成长，朱红 = 危险，金色 = 成就

## 3. 配色方案

```css
:root {
  color-scheme: light;
  --bg-page: #f7f1e6;          /* 暖米色：页面底色 */
  --bg-panel: #fffaf2;         /* 暖白色：面板底色 */
  --bg-locked: #f0e8dc;       /* 浅灰米：锁定底色 */
  --text-main: #211b18;        /* 深墨棕：主文字 */
  --text-muted: #70665d;       /* 灰棕：辅助文字 */
  --text-line: #ddd1c2;        /* 浅暖灰：边框 */
  --text-strong: #b9aa98;      /* 中暖灰：按钮边框 */
  --accent: #2d6f62;           /* 青绿：强调、正确、解锁 */
  --accent-light: #edf8f4;     /* 浅青绿：选中背景 */
  --danger: #9d3c45;           /* 朱红：错误、心魔 */
  --danger-light: #fdf2f3;     /* 浅朱红：错误背景 */
  --gold: #c4953a;             /* 金色：成就、掌握、特殊 */
  --gold-light: #fdf6e8;       /* 浅金色：成就背景 */
  --warning: #d4a017;          /* 琥珀：警告、注意 */
  --purple: #6b5b95;           /* 紫色：心魔回廊、特殊 */
  --shadow: 0 12px 32px rgba(55, 43, 31, 0.08);
  --radius: 8px;
}
```

**色彩语义：**

| 场景 | 主色 | 背景 | 文字 |
|---|---|---|---|
| 正常面板 | — | bg-panel | text-main |
| 选中/正确 | accent | accent-light | accent |
| 错误/心魔 | danger | danger-light | danger |
| 成就/掌握 | gold | gold-light | gold |
| 锁定/未解锁 | — | bg-locked | text-muted |
| 警告/注意 | warning | — | warning |
| 心魔回廊 | purple | — | purple |

## 4. 字体排版

**字体栈：**

```css
/* 正文：书卷感 */
font-family: ui-serif, "Songti SC", "Noto Serif CJK SC", "STSong", serif;

/* 数据/代码：清晰 */
font-family: ui-monospace, "SF Mono", "Fira Code", monospace;

/* 标题：强调 */
font-family: ui-serif, "Songti SC", "Noto Serif CJK SC", serif;
```

## 6. "纯文字"视觉增强技巧

### 6.1 框线装饰

用 Unicode 框线字符构建"卷轴"感：

```
╔══════════════════════════════════════╗
║  标题                                ║
╠══════════════════════════════════════╣
║  内容                                ║
╚══════════════════════════════════════╝
```

### 6.2 分隔线装饰

用不同样式的分隔线区分层级：

```
━━━━━━━━━━━━━━  粗分隔（章节间）
──────────────  中分隔（面板间）
┄┄┄┄┄┄┄┄┄┄┄┄  细分隔（内容间）
```

### 6.3 进度条

用 Unicode 块字符构建：

```
进度: ████████░░ 80%
进度: █████▓░░░░ 55%
进度: ██░░░░░░░░ 20%
```

**CSS 动画：** 宽度从 0 过渡到目标值，背景色渐变。

### 6.4 星级评分

```
难度: ★★★☆☆（3/5）
掌握: ⭐⭐⭐⭐☆（4/5）
```
