# 小明书院：文字巡游 — 美术素材规范

> 文档定位：本游戏为零图片资源项目。所有"视觉"由 CSS 排版和文字渲染完成。本文档定义配色、字体、排版、间距等"纯文字美术"规范，而非图片素材清单。

---

## 1. 设计原则

1. **零图片**：不生成、不引用、不依赖任何图片文件
2. **CSS即美术**：所有视觉效果通过 CSS 变量、边框、阴影、背景渐变实现
3. **文字即角色**：角色没有头像/立绘，仅以名称和台词呈现
4. **结构即氛围**：面板、网格、进度条、日志等布局组件本身就构成游戏感

---

## 2. 配色方案

```css
:root {
  color-scheme: light;
  --text-bg: #f7f1e6;          /* 页面底色：暖米色 */
  --text-paper: #fffaf2;       /* 面板底色：暖白色 */
  --text-ink: #211b18;         /* 主文字：深墨棕 */
  --text-muted: #70665d;       /* 辅助文字：灰棕 */
  --text-line: #ddd1c2;        /* 边框：浅暖灰 */
  --text-strong-line: #b9aa98; /* 按钮边框：中暖灰 */
  --text-accent: #2d6f62;      /* 强调色：青绿 */
  --text-danger: #9d3c45;      /* 错误色：朱红 */
  --text-focus: rgba(45, 111, 98, 0.22);
  --text-shadow: 0 12px 32px rgba(55, 43, 31, 0.08);
  --radius: 8px;
}
```

**配色逻辑：**
- 暖色背景（米/白）→ 营造纸张/书卷感
- 深墨文字 → 模拟墨水
- 青绿强调 → 代表书院/正确/成长
- 朱红错误 → 代表危险/答错/心魔
- 不使用任何蓝色、紫色、渐变色

## 3. 字体排版

**字体栈：** `ui-serif, "Songti SC", "Noto Serif CJK SC", "STSong", serif`

**字号层级：**

| 层级 | 大小 | 字重 | 用途 | 代码 |
|---|---|---|---|---|
| 品牌标题 | 1.35rem | 正常 | 书院名称 | h1 |
| 场景标题 | clamp(1.35rem, 2vw, 1.9rem) | 正常 | 页面标题 | h2 |
| 面板标题 | 默认 | 正常 | 面板标题 | h3 |
| 正文 | 默认 | 正常 | 题干、台词、解析 | p |
| 按钮标签 | 默认 | 粗体 | 按钮主文字 | strong |
| 按钮描述 | 默认 | 正常 | 按钮小字 | small |
| 标签 | 0.82rem | 正常 | 统计标签 | span |
| 日志 | 默认 | 正常 | 状态提示 | muted p |

## 6. "纯文字"视觉增强技巧

### 6.1 面板系统
通过边框、背景、阴影构建"卷轴"感：
- 所有面板有 `border: 1px solid var(--text-line)`
- 所有面板有 `background: var(--text-paper)`
- 所有面板有 `box-shadow: var(--text-shadow)`

### 6.2 状态反馈
- **选中：** 边框变青绿 + 背景变浅绿
- **正确：** 边框变青绿
- **错误：** 边框变朱红
- **悬停：** 边框变青绿（仅按钮）
- **焦点：** 3px 青绿色 outline

### 6.3 进度条
用 CSS 变量控制宽度，实现数值可视化：
```css
.text-meter i {
  width: var(--value);  /* 由 JS 设置百分比 */
  height: 100%;
  background: var(--text-accent);
}
```

### 6.4 日志/历史
底部固定日志区域，用分隔线和 muted 色营造"卷宗记录"感。
