# 小明书院：文字视觉规范

## 1. 视觉定位

小明书院的视觉是文字游戏视觉，不是重美术视觉。

视觉目标：

- 让题干更易读。
- 让选项更好点。
- 让对错反馈更明确。
- 让心魔和学习报告有游戏状态感。

所有视觉元素都服务阅读和作答，不承担核心学习信息。

## 2. 文字游戏感

游戏感来自排版和状态，而不是大图。

可使用：

- 稳定的题卡。
- 明确的进度条。
- 选项状态色。
- 心魔压力标签。
- 学习报告摘要。
- 少量符号。
- 轻动效。

不依赖：

- 人物大图。
- 场景背景。
- 大面积插画。
- 动画演出。
- 素材收集墙。

## 3. 色彩语义

| 状态 | 用途 |
|---|---|
| 主色 | 当前推荐、主按钮、答对状态 |
| 危险色 | 答错、心魔压力、复燃 |
| 警告色 | 强攻风险、连续错误 |
| 中性色 | 普通题干、选项、说明 |
| 奖励色 | 目标完成、净化成功 |

颜色必须和文字同时出现，不能只靠颜色表达状态。

## 4. 排版

| 场景 | 要求 |
|---|---|
| 题干 | 行高充足，优先可读 |
| 选项 | 点击区域稳定，不因选中状态跳动 |
| 按钮 | 文案短，动作明确 |
| 题眼短课 | 1 到 3 句，默认不挤压选项 |
| 学习报告 | 信息分组清晰，下一步按钮突出 |

## 5. 动效

动效只用于反馈，不用于装饰。

| 动效 | 触发 |
|---|---|
| 答对高亮 | 提交正确答案 |
| 答错轻震 | 提交错误答案 |
| 心魔压力变化 | 生成、强化、净化心魔 |
| 报告数字递增 | 进入学习报告 |

所有动效必须支持 reduced motion。

## 6. 图形与图标

图形只可作为辅助：

- 破招图标。
- 心魔压力标记。
- 题阵进度。
- 学习域识别。

图形不能承载题眼、答案、错因或下一步建议。

## 7. 视觉禁区

- 用大图遮挡题干。
- 把学习报告做成装饰展示页。
- 把心魔做成需要识别造型的对象。
- 首屏展示素材墙。
- 让视觉风格比真题内容更突出。
