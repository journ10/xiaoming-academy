# 小明书院：真题题阵 — 游戏形态总纲

## 1. 产品定位

小明书院是一款文字版真题答题游戏。

它保留历年真题的原题干、原选项和原作答结构，用短局、目标、破招、题眼短课、心魔复盘和学习报告，把刷题组织成可持续游玩的备考循环。

它不是视觉 RPG，不是考试模拟器，也不是把真题改造成另一种题型的小游戏。玩家始终面对真实选项作答，游戏层只负责让每局更有目标、更有反馈、更容易复盘。

## 2. 不可破坏的题目原则

- 真题题干不改写。
- 真题选项不改写。
- 单选、多选、判断、案例等原作答结构不改。
- 不要求玩家答前手动标题眼。
- 不使用新互动替代 A/B/C/D/E 等原选项作答。
- 正确率、掌握度、心魔和学习报告都基于真实作答结果。

## 3. 玩家主循环

```text
开局台
  -> 接受或更换本局目标
  -> 选择流派
  -> 进入 5 题题阵
  -> 每题选择破招并按原选项作答
  -> 答后查看题眼短课
  -> 心魔生成、强化或净化
  -> 学习报告
  -> 下一局建议
```

玩家打开游戏后不需要先浏览题库。系统根据近期错因、覆盖度和稳定度给出一个可直接开始的推荐题阵。

## 4. 开局台

开局台只解决一个问题：现在最该打一局什么题阵。

示例：

```text
今日推荐：净化“审题失误”心魔

你最近连续漏看题干限定词。
建议使用“复盘”流派进入 5 题题阵。

本局目标：净化 2 个审题失误心魔
预计时长：4 分钟

[进入题阵]
[换目标] [查看心魔] [学习报告]
```

开局台必须展示：

- 今日推荐。
- 推荐理由。
- 本局目标。
- 推荐流派。
- 进入题阵的主按钮。
- 心魔和学习报告的次级入口。

开局台不展示题库审计信息、完整知识树或长期面板。

## 5. 题阵

一局题阵固定 5 道真题，控制在 3 到 5 分钟。

| 题位 | 作用 |
|---|---|
| 第 1 题 | 进入状态，降低开局摩擦 |
| 第 2 题 | 贴近本局目标 |
| 第 3 题 | 确认当前错因是否存在 |
| 第 4 题 | 增加压力，可能出现多选、判断或案例题 |
| 第 5 题 | 收束题，决定本局报告倾向 |

题阵不是随机题串。每局必须有一个目标，并且结算时说明目标完成情况。

## 6. 题阵类型

| 题阵 | 作用 | 推荐时机 | 结算重点 |
|---|---|---|---|
| 拓新题阵 | 接触新题，扩大覆盖面 | 新题覆盖不足时 | 新增题眼、是否生成心魔 |
| 净魔题阵 | 处理错题和重复错因 | 有活跃心魔时 | 净化数量、是否复燃 |
| 冲刺题阵 | 混合题型和学习域 | 需要综合检验时 | 正确率、薄弱域、错因分布 |

系统默认推荐最适合当前状态的一局，玩家可以更换目标。

## 7. 流派

流派是局前学习策略，不是角色职业或复杂养成。

| 流派 | 含义 | 体验 |
|---|---|---|
| 稳修 | 稳定推进，容错更高 | 适合日常学习或状态一般时 |
| 突击 | 连对收益更高，答错代价更明显 | 适合熟题冲刺 |
| 复盘 | 错题和心魔收益更高 | 适合查漏补缺 |

流派影响本局收益、破招代价和心魔处理效率，不影响题目内容。

## 8. 破招

破招是每题前的轻策略选择。它不替玩家作答，不改变真题结构。

| 破招 | 效果 | 代价 |
|---|---|---|
| 稳破 | 正常作答，正常收益 | 无 |
| 强攻 | 答对收益提高 | 答错时心魔压力更高 |
| 观照 | 先看一句提示再作答 | 本题收益降低 |

破招表达玩家本题的学习态度：稳妥推进、确认后冲刺，或承认不确定先理解。

## 9. 单题体验

单题界面保持考试感：

```text
净魔题阵 · 第 3 / 5 题
目标：净化审题失误心魔

题干：……

A. ……
B. ……
C. ……
D. ……

破招：[稳破] [强攻] [观照]

[提交答案]
```

提交后立刻进入答后反馈。

```text
答错。
正确答案：B
你的答案：C

题眼短课：
本题考查义务教育阶段随迁子女入学保障。
题眼是“非户籍所在地工作或者居住”和“当地人民政府应当”。
你这次的问题是把泛泛的“保障受教育”当成了法条原意。

心魔变化：
审题失误心魔 +1
法规主体心魔出现
```

## 10. 题眼短课

题眼短课是答后短解释，不是独立课程入口。

它固定回答三件事：

- 考点：这题考什么。
- 题眼：题干或选项里最关键的限定。
- 错因：这类题最容易错在哪里。

默认 1 到 3 句，完整解析可展开，但不打断题阵节奏。

## 11. 心魔

心魔是错因实体化。它不是失败惩罚，而是一个还没有稳定解决的问题。

常见心魔：

- 概念混淆
- 审题失误
- 记忆盲区
- 应用失误
- 多选漏选
- 过度自信

规则：

- 答错会生成或强化心魔。
- 同类错误反复出现，心魔压力上升。
- 在净魔题阵中再次答对同题或同类题，心魔压力下降。
- 只看解析不能净化心魔；必须通过再次作答验证。

## 12. 学习报告

学习报告不是成绩单，而是下一局导航。

示例：

```text
本局报告

净魔题阵 · 复盘流派
正确率：3 / 5
本局目标：净化 2 个审题失误心魔
完成情况：1 / 2

新增心魔：法规主体心魔
净化心魔：审题失误心魔 -1
主要问题：不是知识空白，而是限定词漏看。

下一局建议：
继续净魔题阵，保留复盘流派。
优先处理“法规主体”和“审题失误”的交叉题。
```

报告必须给出可执行的下一步。

## 13. 长期成长

长期成长不靠装饰收集，而靠学习状态变清晰。

六个学习域：

- 学习心理与认知机制
- 教育学原理、课程与教学
- 教育法律法规与政策制度
- 德育、班级管理与家校协同
- 学生身心发展与个体差异
- 教师职业素养与专业规范

每个学习域展示三种状态：

- 覆盖度：接触了多少题。
- 稳定度：最近是否能稳定答对。
- 心魔压力：主要错因是否还在反复出现。

玩家长期追求的不是通关某个区域，而是让薄弱点越来越明确、心魔越来越少、答题越来越稳定。

## 14. 成功标准

- 玩家 10 秒内能知道该点哪里开始。
- 一局结束后能知道自己哪里错、为什么错、下一局该做什么。
- 真题作答结构保持可信。
- 错题能沉淀为可复测、可净化的心魔。
- 连续数局后，玩家感觉自己在推进备考，而不是浏览题库。
