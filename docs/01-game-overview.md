# 小明书院：秘卷巡游 — 游戏设计总纲

> 当前基准：本项目已经从“七章地图式 RPG”改为“Roguelite 备考学习游戏”。旧的章节主线、独立剧情页和“综合知识”终章不再作为产品方向；完整改造依据见 `docs/11-roguelite-transformation-plan.md`。

## 1. 设计目标

| 目标 | 标准 | 验证方式 |
|---|---|---|
| 真会题 | 玩家必须反复接触真题、解析、错因和复测 | 心魔净化后复测正确率、薄弱域正确率 |
| 好开始 | 玩家打开后知道第一步做什么 | 首屏点击“开始一局”的比例 |
| 有娱乐性 | 每局有目标、风险、奖励和下一局选择 | 连续游玩局数、模式/流派使用分布 |
| 不混乱 | 游戏内容和题库审计信息分层 | 首屏无源题位、复核数、题位等审计指标 |
| 可持续 | 4374 道当前可玩题能支撑长期备考 | 题库覆盖率、日课完成率、心魔清理率 |

## 2. 核心体验

玩家不是“进地图找章节”，而是持续完成一局一局的学习题阵：

```text
开局台推荐目标
  -> 选择本局模式
  -> 选择学习流派
  -> 5 题题阵
  -> 即时对错与错因
  -> 战报
  -> 下一局建议
```

每局必须回答三个问题：

- 这局要解决什么学习问题？
- 这局用什么策略？
- 结束后下一步做什么？

## 3. 三种单局模式

| 模式 | 使用时机 | 题目选择 | 学习价值 | 娱乐性 |
|---|---|---|---|---|
| 探索局 | 新题和新知识域还很多 | 优先未答题、低掌握题 | 扩大覆盖面 | 开地图、拿新节点 |
| 净化局 | 有活跃心魔或重复错题 | 优先错题、心魔题、低正确率题 | 纠错和复测 | 清债、削弱心魔 |
| 冲刺局 | 需要综合模拟或临考手感 | 混合多个学习域与题型 | 切换训练 | 连破、限资源压力 |

推荐规则：

- 有活跃心魔时，默认推荐净化局。
- 没有心魔且仍有大量未答题时，默认推荐探索局。
- 覆盖率较高或需要混合训练时，推荐冲刺局。

## 4. 三种起始流派

| 流派 | 效果定位 | 适合玩家状态 |
|---|---|---|
| 稳修 | 容错更高、收益稳定 | 不确定从哪里开始、日常推进 |
| 突击 | 连破收益更高、答错代价更明显 | 短时间冲刺、想要刺激感 |
| 复盘 | 心魔和错题收益更高、新题推进较慢 | 错题债较多、考前查漏 |

流派不是装饰，而是元认知选择：玩家需要判断自己今天该稳定推进、集中冲刺，还是回头修错。

## 5. 内容域

重新分类后，题库采用六个真实学习域 + 待人工归类，不再使用“综合知识”作为兜底章。

| 学习域 | 题数 |
|---|---:|
| 学习心理与认知机制 | 1144 |
| 教育学原理、课程与教学 | 990 |
| 教育法律法规与政策制度 | 768 |
| 德育、班级管理与家校协同 | 638 |
| 学生身心发展与个体差异 | 507 |
| 教师职业素养与专业规范 | 213 |
| 待人工归类 | 114 |

题库审计口径：

- PDF 源题位：4680。
- 当前可玩题：4374。
- 需人工归类：114。
- OCR/分类质量信息保留在审计文件和人工复核清单中，不作为首屏游戏内容展示。

## 6. 心魔系统

心魔代表“还没真正学会的题”，不是失败惩罚。

| 状态 | 触发 | 游戏表现 | 学习动作 |
|---|---|---|---|
| 生成 | 答错或低置信误判 | 出现在心魔页和战报 | 记录错因 |
| 强化 | 同类错误重复出现 | 压力上升 | 提醒优先净化 |
| 净化 | 净化局或复测答对 | 压力下降或移除 | 验证已经会题 |
| 复燃 | 净化后再次错 | 重新激活 | 加入下一局推荐 |

错误类型保留四类：概念混淆、审题失误、记忆盲区、应用失误。

## 7. 叙事原则

剧情保留，但不再是独立入口。任何叙事文本必须绑定至少一个作用：

- 说明本局目标。
- 解释遭遇变化。
- 解释错因或心魔。
- 强化战报里的学习结论。

如果一段剧情只是“点进去看几句话”，它不进入主流程。

## 8. 首屏原则

首屏只解决“从哪里开始”：

- 当前推荐局。
- 本局目标。
- 推荐流派。
- 开始一局。
- 换目标。
- 去心魔或学习报告的次级入口。

首屏不展示：

- PDF 源题位。
- OCR 复核数。
- 待归类数量。
- 完整知识图谱。
- 七章路线。
- 独立剧情按钮。

## 9. 成功标准

玩家首次进入 10 秒内能开始第一局；一局结束后能看懂自己哪里错、下一局为什么被推荐；连续数局后，玩家能感觉自己不是在浏览题库，而是在推进一个可反复游玩的备考循环。
