# 《小明书院：秘卷巡游》重新设计方案

## 一、现状诊断

### 1.1 现有架构速览

| 维度 | 当前状态 |
|---|---|
| 游戏类型 | 静态网页轻量RPG，用游戏化包装教招真题刷题 |
| 核心循环 | 世界地图 → 剧情对话 → 短课练功 → 题阵战斗 → 心魔净化 → 章节通关 |
| 角色系统 | 4名角色（明澈、阿芷、青岚、小墨），各负责序章/练功/战斗/心魔 |
| 战斗系统 | 3种破招式（稳破/强攻/观照）× 4类路线节点（常阵/锐阵/息阵/宝阵） |
| 成长系统 | 心力、灵页、星辉、羁绊、心法熟练、等级 |
| 美术资源 | 世界地图、战斗背景、4角色头像+立绘、SVG辅助符号 |
| 技术栈 | 原生HTML/CSS/JS，无框架，无后端，localStorage存档 |

### 1.2 核心问题

**剧情层面：**
- 角色对话过于功能性，更像"玩法说明书"而非故事
- 缺乏真正的戏剧冲突和悬念，六章的"修复秘卷"目标感偏弱
- 角色之间没有互动关系，每个人都是独立的功能NPC
- 没有情感弧线，玩家只是"过客"而非"参与者"

**玩法层面：**
- 破招式数值差异被感知度不足，玩家容易固定只选一种
- 路线节点选择影响有限，4种类型不足以形成有意义的策略空间
- 练功和战斗的衔接不够紧密，练功后战斗检验的反馈感不强
- 心魔系统缺乏压迫感，净化过程过于简单
- 缺乏"局内Build"感，每局5题之间没有形成连贯的决策链

**UI层面：**
- 战斗界面信息层级不够清晰，题干和选项的"敌人感"不够强
- 结算报告缺乏视觉冲击力，成长反馈像数据报表
- 世界地图节点缺乏"待探索"和"已通关"的强状态区分
- 视觉小说对话框表现力不足，缺少表情变化、场景切换等叙事手法
- 缺少局内特效（正确/错误的视觉反馈、连击提示、心魔出现动画等）

**美术层面：**
- 心魔没有专属形象，只有名称标签
- 材料/掉落物没有图标，只有文字
- 心法系统没有视觉符号
- 路线节点没有图标区分
- 章节封印没有解锁动画
- 缺少战斗特效和过渡动画素材

## 二、剧情重新设计

### 2.1 核心叙事升级：从"修复秘卷"到"真相巡游"

**新主线悬念：**
> 小明书院不是一所普通的学校——它是一座"显化书院"，能把人的知识变成实体秘境。但最近，书院的"真题秘卷"被一种名为"遗忘之雾"的力量污染，不仅题目变成了心魔，连书院本身也在逐渐褪色。玩家作为新一届"巡游者"，必须和四位同伴一起，在修复秘卷的过程中，逐步揭开"遗忘之雾"的源头——原来，这是一场来自书院内部某位前辈的"试炼"……

**叙事结构：六章真相揭露**

| 章节 | 叙事主题 | 剧情推进 | 悬念揭示 |
|---|---|---|---|
| 序章 | 书院入门 | 明澈介绍书院来历，玩家被选中为巡游者 | 书院最近有秘卷失窃事件 |
| 第一章 | 律令花窗 | 修复教育法规封印，阿芷开始陪伴练功 | 发现失窃的秘卷边缘有奇怪的黑色墨迹 |
| 第二章 | 观心花园 | 进入教育心理学秘境，青岚加入战斗 | 黑色墨迹会"说话"，似乎在提醒什么 |
| 第三章 | 授业工坊 | 教学设计题阵中，遇到第一个"异常心魔" | 心魔不是外敌，而是书院前辈的"善意投影" |
| 第四章 | 师德星廊 | 教师职业道德试炼，小墨揭示心魔回廊的秘密 | 前辈是书院创始人，用遗忘之雾测试新一代巡游者 |
| 第五章 | 统御回廊 | 班级管理秘境，四人开始合作 | 前辈说：真正的试炼不是答对题目，而是学会面对错题 |
| 第六章 | 童心星谷 | 最终章，儿童发展秘境，全库通关触发 | 前辈现身："我当年也是被心魔困住的人，现在书院交给你了" |

### 2.2 角色关系网重设计

从"各自负责一个功能"升级为"有情感互动的小团队"：

**明澈（书院导师）→ "引路人"**
- 表面：冷静、理性、负责序章和战报
- 秘密：他是创始人前辈的弟子，知道遗忘之雾的真相，但选择在合适时机才揭示
- 与玩家关系：亦师亦友，随着章节推进，会从"教导"变成"请教"
- 情感弧线：从"我必须保护新巡游者"到"你比我当年更勇敢"

**阿芷（练功同伴）→ "同行者"**
- 表面：温柔、细心、负责题眼讲解
- 秘密：她曾经是"错题最多"的巡游者，差点被心魔吞噬，是小墨救了她
- 与玩家关系：最能理解"答错不可怕"的同伴，每次净化心魔后她会特别开心
- 情感弧线：从"我担心你也会失败"到"我为你骄傲"

**青岚（破阵同伴）→ "挑战者"**
- 表面：自信、好胜、负责战斗鼓励
- 秘密：她的连破记录是书院最高，但她自己也有一个从未净化的心魔（"过度自信"）
- 与玩家关系：竞争中的合作者，如果玩家连破超过她，她会"恼羞成怒"地祝贺
- 情感弧线：从"我来帮你"到"你帮我看到了我的盲区"

**小墨（秘卷灵）→ "守护者"**
- 表面：神秘、寡言、负责心魔回廊
- 秘密：她就是书院创始人前辈的分身，一直在暗中观察玩家
- 与玩家关系：从"旁观者"到"托付者"
- 情感弧线：最后一章揭示身份，将书院交给玩家守护

### 2.3 每章剧情脚本框架（示例：第一章完整对话）

**第一章 律令花窗 - 开场剧情**

```
[场景：书院正门，傍晚，灯笼亮起]

明澈："欢迎来到小明书院。从今天起，你就是一名巡游者了。"
明澈："书院的真题秘卷是历代考生心血的结晶。但最近，秘卷上的字迹正在褪色。"
明澈："我们称之为'遗忘之雾'。它会污染知识，把题目变成心魔。"

[阿芷登场，从画面左侧出现]

阿芷："明澈前辈说得有点吓人啦。其实，遗忘之雾也没那么可怕。"
阿芷："我第一次当巡游者的时候，心魔回廊里有二十多个错题心魔呢。"
阿芷："但你看，我现在不是好好的？所以别担心，我会陪你一起。"

[明澈微笑]

明澈："阿芷说得对。第一步，我们先修复最外层的封印——'律令花窗'。"
明澈："教育法规是书院的基础，也是秘卷的第一层防护。"

[画面切换：进入律令花窗地图]

阿芷："我先帮你把题眼整理成短课。记住关键词后，青岚会带你进题阵战斗。"
阿芷："答对的话，封印就会亮起来。答错的话……也没关系，心魔是可以净化的。"
```

**第一章 律令花窗 - 通关剧情**

```
[场景：花窗封印亮起，蓝色光芒]

青岚："漂亮！第一道封印亮了！我记录下来了——书院最快通关纪录保持者还是我吧？"

[小墨从阴影中出现]

小墨："……我注意到，失窃的秘卷边缘，有黑色墨迹渗到了这一章。"
小墨："不是遗忘之雾。是别的东西。它在……写字？"

[玩家看到一段模糊的墨迹文字："面对错题，不要逃避"]

明澈："这不可能。遗忘之雾不会写字。"
明澈："巡游者，这件事暂时不要声张。我们继续修复下一章，同时留意这些墨迹。"

阿芷："不管发生了什么，我们一起面对。下一章是观心花园，我先去准备短课。"

[第一章通关，解锁第二章，同时在地图上显示一个"黑色墨迹"标记]
```

### 2.4 全六章悬念节奏

采用"三幕式"结构，六章分为两组三幕：

**第一幕：入门（第1-2章）**
- 建立世界观：书院、秘卷、巡游者、遗忘之雾
- 建立角色关系：明澈引导、阿芷陪伴、青岚加入
- 引入异常：黑色墨迹首次出现

**第二幕：探索（第3-4章）**
- 异常升级：心魔开始"说话"，不是简单的错题，而是带有个性化信息
- 角色秘密揭露：小墨的观察、阿芷的过去、青岚的盲区
- 玩家选择：剧情对话中开始出现"你更倾向于哪种处理方式？"的轻量选择（不影响主线，但影响角色对话细节）

**第三幕：真相（第5-6章）**
- 真相揭露：遗忘之雾是创始人前辈留下的"试炼机制"
- 高潮：第六章最终战斗，需要同时面对所有主题的心魔混合题阵
- 结局：全库通关后，前辈现身，书院传承

## 三、玩法重新设计

### 3.1 核心循环保留与优化

保留现有核心循环的框架：
```
世界地图 → 剧情对话 → 短课练功 → 题阵战斗 → 心魔净化 → 章节通关
```

但在每个环节增加"决策深度"和"策略反馈"：

### 3.2 破招式系统升级：连携与精通

**现有问题：** 三种招式（稳破/强攻/观照）只是数值倍率，玩家感受不到"精通"的快感。

**新设计：招式精通与连携**

| 招式 | 基础效果 | 精通效果（使用该招式10次后解锁） | 连携效果（连续使用同招式2次） |
|---|---|---|---|
| 稳破 | 正常伤害/收益，低失误代价 | 稳如磐石：失误时只损失50%心力 | 连续稳破第3次：额外恢复1点心力 |
| 强攻 | 高伤害/高收益，高失误代价 | 破釜沉舟：答对时连破计数+2 | 连续强攻第2次：伤害倍率额外+0.3 |
| 观照 | 显示题眼提示，降低收益 | 慧眼识珠：提示不再降低收益 | 连续观照：提示质量升级（从"题眼"到"排除一个错误选项"） |

**UI变化：** 在招式按钮上增加"精通进度条"和"当前连携状态"，让玩家能直观看到自己的"招式熟练度"。

### 3.3 路线节点系统升级：分支与预判

**现有问题：** 4种节点类型不够丰富，路线选择策略空间有限。

**新设计：7种节点 + 预判机制**

**新增节点类型：**

| 节点 | 图标 | 效果 | 策略价值 |
|---|---|---|---|
| 常阵 | 普通卷轴 | 标准战斗 | 安全选择 |
| 锐阵 | 带刺卷轴 | 高收益高惩罚 | 自信时选择 |
| 息阵 | 莲花卷轴 | 低收益，恢复心力 | 心力低时选择 |
| 宝阵 | 金色卷轴 | 灵页奖励翻倍 | 缺资源时选择 |
| 心魔回廊 | 黑色卷轴 | 复训错题 | 有错题时选择 |
| **谜阵** | 紫色卷轴 | 题目难度+1，但奖励额外+50% | 想赌高难度时选择 |
| **共鸣阵** | 彩虹卷轴 | 同主题连续答对有额外加成 | 某主题熟练度高时选择 |
| **试炼阵** | 红色卷轴 | 必须答对，答错直接扣除2点心力 | 冒险型玩家选择 |

**预判机制：** 在进入路线时，玩家可以看到每个节点的"类型"和"主题方向"（如"教育法规·锐阵"），但看不到具体题目。这允许玩家根据当前状态（心力、主题熟练度、错题情况）做出策略选择。

### 3.4 心魔系统升级：个性化与压迫感

**现有问题：** 心魔只是错题的标签，没有形象和压迫感。

**新设计：具象化心魔**

每个主题的心魔有专属形象、攻击方式和净化机制：

| 主题 | 心魔名称 | 形象 | 攻击方式 | 净化机制 |
|---|---|---|---|---|
| 教育法规 | 律令心魔 | 身穿黑色官袍的模糊人影 | 用"法条迷雾"遮挡选项关键词 | 连续答对2次：迷雾消散 |
| 教育心理学 | 观心心魔 | 漂浮的破碎镜子 | 制造"认知干扰"（选项顺序随机打乱） | 连续答对2次：镜子复原 |
| 教学设计 | 授业心魔 | 断裂的教鞭和散落的教案 | 增加答题时间压力（倒计时） | 连续答对2次：教案重聚 |
| 教师职业道德 | 师德心魔 | 双面面具，不断切换表情 | 正确答案和错误答案外观相似 | 连续答对2次：面具脱落 |
| 班级管理 | 统御心魔 | 混乱的班级剪影 | 增加干扰选项（从4个变5个） | 连续答对2次：班级安静 |
| 儿童发展 | 童心心魔 | 哭泣的孩童幻影 | 减少心力上限（临时-1） | 连续答对2次：孩童微笑 |

**心魔压迫机制：** 当活跃心魔数量超过3个时，全属性降低5%；超过5个时，降低10%。这创造了一种"被心魔包围"的紧迫感。

### 3.5 心法共鸣系统（新增）

**现有问题：** 心法熟练度只是一个百分比，没有改变玩法。

**新设计：心法等级改变战斗体验**

| 主题 | 心法等级 | 效果 |
|---|---|---|
| 教育法规 | 1-3级 | 无特殊效果 |
| 教育法规 | 4-6级 | 稳破失误时，心魔压迫-20% |
| 教育法规 | 7-10级 | 该主题题阵中，首题自动显示一个题眼提示 |
| 教育心理学 | 4-6级 | 观照提示不再降低收益 |
| 教育心理学 | 7-10级 | 该主题题阵中，连破计数从2开始（而非0） |
| 教学设计 | 4-6级 | 强攻收益倍率+0.2 |
| 教学设计 | 7-10级 | 该主题题阵中，答错不触发心魔生成（只强化现有） |
| 教师职业道德 | 4-6级 | 净化心魔时，额外获得灵页 |
| 教师职业道德 | 7-10级 | 该主题题阵中，每答对一题恢复0.5心力 |
| 班级管理 | 4-6级 | 息阵恢复心力+1 |
| 班级管理 | 7-10级 | 该主题题阵中，初始心力+2 |
| 儿童发展 | 4-6级 | 所有心魔压迫增长速度-25% |
| 儿童发展 | 7-10级 | 全主题题阵中，最终题答案视为"已练功"（即正确时获得额外奖励） |

### 3.6 每日巡游与限时挑战（新增）

**每日巡游：** 每天刷新一条特殊路线，包含5个主题混合的节点。完成可获得"今日运势"奖励（随机材料掉落）。

**限时挑战：** 每周解锁一个"限时秘境"（如"3分钟内答对10题"或"全程只许用强攻通关"），完成获得限定称号。

### 3.7 数值平衡调整

| 参数 | 当前 | 建议调整 |
|---|---|---|
| 初始心力 | 6 | 保持6 |
| 每题稳破答错损失 | 1心力 | 保持1 |
| 每题强攻答错损失 | 2心力 | 调整为2.5（增加风险感） |
| 观照收益倍率 | 0.58 | 调整为0.65（略微提高，鼓励使用） |
| 息阵恢复 | 1心力 | 保持1 |
| 章节心法门槛 | 题目数×14% | 调整为题目数×12%（略微降低门槛） |
| 连破计数上限 | 无 | 每局最多10连破（超过后不再增加，但保留当前奖励） |
| 心魔压迫上限 | 无 | 单局心魔压迫达到5时，强制结束并进入"心魔暴走"结算 |

## 四、UI重新设计

### 4.1 设计原则

1. **游戏优先**：每一个界面第一眼必须是游戏，不是题库
2. **反馈即时**：每个操作都有视觉/动画反馈，延迟不超过200ms
3. **信息分层**：核心信息（题目/选项）最大，次要信息（状态/奖励）辅助
4. **叙事融入**：UI元素本身承载叙事功能（如心魔形象、封印动画等）
5. **低饱和度国风**：保留现有视觉方向（钴蓝、青绿、琥珀金、朱红），但增加更多层次

### 4.2 世界地图界面重设计

**当前问题：** 地图节点较小，状态区分不够明显，角色立绘和地图的融合度不高。

│         │   小明书院   │                        │

│              │                                   │
