# 小明书院：秘卷巡游 — 事件化叙事设计

> 当前基准：叙事不再是七章剧情页，也不再是“读完几句话”的进度门槛。剧情只作为开局、遭遇、答错、净化、结算中的事件反馈存在。

## 1. 叙事原则

| 原则 | 说明 |
|---|---|
| 服务玩法 | 剧情必须改变或解释本局目标、规则、奖励、错因、下一步 |
| 服务学习 | 角色台词要帮助玩家理解“为什么错”和“该怎么补” |
| 短而准 | 单段文本优先 1-3 句，不打断刷题节奏 |
| 不做门槛 | 不要求玩家先看剧情才能答题 |
| 不做章节包装 | 六个学习域是题池和机制来源，不是剧情关卡 |

## 2. 角色职责

| 角色 | 当前职责 | 触发位置 | 文案方向 |
|---|---|---|---|
| 明澈 | 给出学习目标和稳定建议 | 开局台、探索局、报告 | “先把范围铺开，知道自己会什么” |
| 阿芷 | 解释错因、心魔和复盘路径 | 答错、净化局、心魔页 | “错题不是羞辱，是待修复的入口” |
| 青岚 | 强化冲刺、连破、风险选择 | 冲刺局、突击流派、连对 | “快可以，但要知道自己在赌什么” |
| 小墨 | 归纳长期进度和下一步 | 战报、学习报告、日课 | “下一局该补哪块” |

角色不承担独立剧情页功能；角色是反馈系统的人格化外壳。

## 3. 叙事出现位置

### 3.1 开局台

用于说明今天为什么推荐这局。

示例：

```text
阿芷把错题签摊开：这几题又亮了，先别急着开新路，今天把它们压下去。
```

必须同时展示可行动目标，例如“净化 5 题心魔”。

### 3.2 模式选择

用于解释模式差异，不写背景故事。

```text
探索局：点亮没见过的知识点。
净化局：处理已经暴露的问题。
冲刺局：训练跨域切换和压力下判断。
```

### 3.3 流派选择

用于帮助玩家做学习策略判断。

```text
稳修不是保守，是把今天的收益做稳。
突击适合短时冲刺，但答错会更疼。
复盘会放慢新题推进，换来更高的错题收益。
```

### 3.4 局内遭遇

用于解释当前题为什么出现。

```text
这是本局第 3 遭遇：同类概念连续出现，系统正在确认你是否真的区分开了。
```

### 3.5 答错和心魔

用于把失败翻译成学习任务。

```text
阿芷：你不是不会这章，而是把两个相近概念当成了同一个词。下一次先找差异。
```

### 3.6 战报

用于总结本局价值和下一局选择。

```text
小墨：这局新增 3 个知识点，但也暴露了 1 个审题心魔。下一局建议净化，不要让它复燃。
```

## 4. 禁止出现的叙事形态

- 独立“剧情”主入口。
- “完成剧情”按钮。
- 必须读完剧情才能解锁题目。
- 七章开场/通关剧情。
- 与规则、学习反馈、奖励无关的世界观长段落。

## 5. 事件模板

### 探索局开局

```text
明澈：今天先别急着追求全对。你还有一片没走过的题域，本局目标是把新题暴露出来。
```

### 净化局开局

```text
阿芷：心魔不是失败记录，是复测清单。我们只处理最该处理的几道题。
```

### 冲刺局开局

```text
青岚：这局不让你舒服地待在一个主题里。考试也不会提醒你下一题考什么。
```

### 连对反馈

```text
青岚：连破成立，但别让速度替你做决定。下一题先看题干限定词。
```

### 连错反馈

```text
阿芷：暂停一下，同类错误正在重复。先看解析里的关键词，再继续。
```

### 净化成功

```text
小墨：这道题从心魔清单移走了。不是因为你记住答案，而是因为你刚刚复现了理由。
```

## 6. 长期叙事

长期叙事不靠章节推进，而靠学习状态变化：

| 里程碑 | 触发 | 叙事作用 |
|---|---|---|
| 第一次答错 | 首个心魔生成 | 降低挫败感，说明心魔机制 |
| 第一次净化 | 心魔压力清零 | 建立“错题可修复”的信念 |
| 首次冲刺连破 | 连续答对 | 强化策略选择和风险意识 |
| 主要错因变化 | 错题画像更新 | 让玩家看见自己真的在进步 |
| 待人工归类处理 | 121 题清单被人工修正后 | 作为内容质量事件，不进入主线 |

## 7. 文案验收

每段叙事上线前必须回答：

1. 它在哪个操作后出现？
2. 它解释了什么学习状态？
3. 它给玩家下一步了吗？
4. 删掉它会不会影响玩家理解规则或学习反馈？

第四题如果答案是“不会”，这段文案就不应该出现在主流程。
