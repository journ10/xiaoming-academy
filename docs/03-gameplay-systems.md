> 文档定位：核心玩法的设计蓝图。目标：让玩家在"好玩"的体验中"真学会"。

## 2. 知识图谱系统

### 2.1 概念

每道题是图谱上的一个**节点**。节点之间有**依赖关系**（概念A是概念B的前提）。

**节点状态**：
- 灰色：未解锁（依赖未满足）
- 可点击：已解锁但未答题
- 绿色：已答对
- 红色：答错（有心魔）
- 金色：下游全部点亮，概念"掌握"

### 2.2 示例图谱（教育法规片段）

```
                    [教育法规] ——金色（下游全亮）
                   /          \
        [义务教育法] ——绿色  [教师法] ——绿色
           /      \
    [免试入学] ——绿色  [就近入学] ——绿色
         \
      [政府保障] ——绿色
           \
        [非户籍入学] ——红色（有心魔）
```

### 2.3 玩家体验

- 答对"免试入学" → "就近入学"解锁（从灰变可点击）
- 答对"就近入学" → "政府保障"解锁
- 答错"政府保障" → 节点变红，心魔诞生，下游"非户籍入学"锁定（无法答题）
- 净化"政府保障" → "非户籍入学"解锁
- 全部下游点亮 → "义务教育法"变金色，获得"义务教育法·掌握"成就

### 2.4 学习价值

- 玩家看到**"为什么这道题重要"**——它是某个大概念的一部分
- 玩家看到**"我哪里薄弱"**——红色节点聚集的区域就是薄弱区
- 玩家有**"补漏方向"**——优先解锁灰色节点，扩大知识版图

### 2.5 实现方式（纯文字）

用文本缩进和 Unicode 字符构建树形图：

```
教育法规 · 掌握度 85%
├── 义务教育法 ✓
│   ├── 免试入学 ✓
│   ├── 就近入学 ✓
│   └── 政府保障 ✗（心魔 1）
│       └── 非户籍入学 🔒（需先净化政府保障）
├── 教师法 ✓
│   ├── 教师权利 ✓
│   └── 教师义务 ✓
└── 未成年人保护法 ✓
    └── 学校保护 ✓
```

## 3. 流派 Build 系统

### 3.1 基础流派（初始解锁）

| 流派 | 效果 | 适合 | 学习隐喻 |
|---|---|---|---|
| **均衡派** | 无特殊加成，但容错+1心力，稳定 | 新手/日常 | 稳扎稳打的学习 |
| **律令派** | 法规题收益+50%，非法规题惩罚+30% | 冲刺法规 | 主题专攻 |
| **观心派** | 多选题提示质量+1级，但单选题无加成 | 心理学大章 | 针对性策略 |

### 3.2 进阶流派（解锁条件）

| 流派 | 解锁条件 | 效果 | 学习隐喻 |
|---|---|---|---|
| **突击派** | 连破10次 | 连续答对同主题题时，连破倍率×2 | 主题冲刺 |
| **复盘派** | 净化20个心魔 | 心魔净化收益+100%，新题收益-20% | 错题优先 |
| **速攻派** | 第三章通关 | 答题时间+10秒，但收益+30% | 时间管理 |
| **深读派** | 第四章通关 | 练功时额外获得1个"知识点卡片" | 深度学习 |
| **混沌派** | 第七章通关 | 每局随机获得一个其他流派的效果 | 融会贯通 |

### 3.3 流派选择界面（每局前）

```
选择流派（影响整局5题）

[均衡派]     [律令派]     [观心派]     [突击派]     [复盘派]
容错+1      法规+50%     多选提示+1    连破×2      净化+100%
稳定         专注         精确         冲刺         回头

当前推荐：突击派（你正在冲刺第二章，心理学题多）
```

### 3.4 学习价值

- 玩家理解**"不同的学习任务需要不同的策略"**
- 冲刺考试前用"突击派"，查漏补缺用"复盘派"
- 流派选择本身是**元认知训练**——"我现在该用什么方法学习？"

## 4. 错误模式系统

### 4.1 4种错误类型

| 类型 | 诊断问题 | 典型表现 | 净化策略 | 游戏机制 |
|---|---|---|---|---|
| **概念混淆** | 相似概念分不清 | 正强化/负强化、泛化/分化 | 对比表格+例题区分 | 心魔="镜像"，选项互相替换 |
| **审题失误** | 漏看关键词 | 没看到"不""除外""错误的是" | 审题训练：高亮关键词 | 心魔="迷雾"，遮挡关键词 |
| **记忆盲区** | 完全没学过 | 完全没印象，蒙猜 | 强制先练功再战斗 | 心魔="空洞"，题干缺失 |
| **应用失误** | 知道但不会用 | 知道概念，但情境不匹配 | 案例分析+情境匹配 | 心魔="变形"，情境扭曲 |

### 4.2 错误诊断流程

答错后，弹出诊断对话框：

```
❌ 答错了！

错误诊断：
  题目："正强化是指..."
  你的答案：B
  正确答案：A

错误类型分析：
  ▓▓▓▓▓▓▓▓░░ 80% 概率：概念混淆
    你把"正强化"和"负强化"搞混了。
  ▓▓▓░░░░░░░ 20% 概率：记忆盲区
    可能你还没完全理解这两个概念。

建议：
  → 去练功室对比"正强化 vs 负强化"
  → 或者：直接尝试净化这个心魔

[去练功] [直接净化] [知道了，继续]
```

- 错误模式变化趋势显示**"我真的在进步"**

## 5. 章节差异化机制（7章）

### 5.1 第一章：律令花窗（教育法规，649题）

**核心机制：法条迷雾**

- 判断题占比50%（1269题中649题属于法规，但法规中判断题占比约60%）
- 关键词被"迷雾"遮挡："当地人民政府应当为其提供____接受义务教育的条件"
- 必须先练功记住关键词，否则无法正确选择
- 迷雾程度：难度1=遮挡1词，难度2=遮挡2词，难度3=遮挡3词+选项变形

**学习训练**：精确记忆、关键词识别、措辞辨析

### 5.2 第二章：观心花园（教育心理学，868题）

**核心机制：概念迷宫**

- 多选题占比高（约35%）
- 选项"变形"：相似概念互相替换措辞
- 例：正强化的选项中出现"给予愉快刺激"（对）和"撤销厌恶刺激"（负强化，错）
- 变形程度：Lv1=措辞微调，Lv2=概念互换，Lv3=混合陷阱（一个选项里两个概念各对一半）

**学习训练**：概念精确区分、多选全选能力

### 5.3 第三章：授业工坊（教学设计，811题）

**核心机制：时间沙漏**

- 案例分析占比高（约35%）
- 每题有倒计时（难度1=60秒，难度2=45秒，难度3=30秒）
- 材料长（200-300字），需要快速抓关键
- 息阵节点不恢复心力，而是**增加时间**（+15秒）

**学习训练**：信息快速提取、长材料阅读、时间压力下的决策

### 5.4 第四章：师德星廊（教师职业道德，124题）

**核心机制：道德天平**

- 隐藏属性"师德值"（初始0，范围-10到+10）
- 每题有3种答题风格：理想主义（+2）、平衡主义（+0）、现实主义（-1）
- 师德值>5：后续题"理想选项"极具诱惑（正确率陷阱）
- 师德值<-3：后续题惩罚加重（心力损失+1）
- 通关条件：答完124题 + 师德值在[-2, 3]之间

**学习训练**：情境判断、平衡思维、价值观辨析

### 5.5 第五章：统御回廊（班级管理，367题）

**核心机制：策略选择**

- 论述题/策略题出现（约30%）
- 不是选ABCD，而是选"处理策略"（如"先安抚/先保护/先通知"）
- 选择后触发"后果链"：
  - 即时后果（心力变化）
  - 后续影响（下一题的变化）
  - 统御值积累（解锁秘籍）
- 最优策略：无惩罚，+统御值

**学习训练**：策略思维、顺序决策、后果预判

### 5.6 第六章：童心星谷（儿童发展，260题）

**核心机制：精确记忆**

- 填空题集中（虽然实际只有11题，但机制设计如此）
- 精确匹配：大小写不敏感，但关键词必须一致
- 首次答错→强制心魔→净化后才能再试
- 净化机制：第一次给出"首字母提示"，第二次完全凭记忆

**学习训练**：精确记忆、年龄节点、阶段名称

### 5.7 第七章：万象书阁（综合知识，998题）

**核心机制：万象混沌**

- 每题随机从六章机制中选一种
- 心魔跨主题混合（如"法规措辞心魔"+"心理学概念迷宫"）
- 心法全部生效（取最高等级）
- 流派效果混乱（每题可能应用不同的流派加成）

**学习训练**：知识切换、综合运用、跨主题迁移

## 6. 破招式系统（保留+微调）

| 招式 | 效果 | 适合题型 | 风险 |
|---|---|---|---|
| **稳破** | 正常收益，失误损失1心力 | 判断题、填空题 | 低收益 |
| **强攻** | 收益+50%，失误损失2心力 | 案例分析、论述题 | 高风险高回报 |
| **观照** | 提供提示，收益-30% | 多选题、概念混淆题 | 适合学习阶段 |

**与流派联动**：
- 律令派 + 稳破：法规题收益+50% + 稳破正常收益 = 稳定高收
- 观心派 + 观照：多选题提示+1 + 观照提示 = 几乎无错
- 突击派 + 强攻：连破×2 + 强攻+50% = 超高收益（但答错损失惨重）
