# 小明书院：文字反馈与题眼短课规范

## 1. 文档定位

小明书院是文字版真题答题游戏。文字反馈承担游戏感、学习解释和下一步导航，但不承担长篇叙事或美术包装。

所有文案都必须服务四件事：

- 说明本局目标。
- 解释本题考点。
- 翻译玩家错因。
- 指向下一步行动。

## 2. 文字原则

| 原则 | 要求 |
|---|---|
| 短 | 单段默认 1 到 3 句 |
| 准 | 必须指向题目、错因或下一步 |
| 可行动 | 玩家看完知道接下来该做什么 |
| 不羞辱 | 答错反馈描述问题，不评价玩家 |
| 不喧宾夺主 | 文字不能挡住题干、选项、答案和按钮 |

## 3. 开局台文案

开局台文案回答：

- 推荐什么题阵。
- 为什么推荐。
- 这局目标是什么。
- 建议使用什么流派。

模板：

```text
今日推荐：净化“{心魔类型}”心魔

你最近在「{学习域}」中连续出现「{错因}」。
建议使用「{流派}」流派进入 5 题题阵。

本局目标：{目标}
```

## 4. 局内状态文案

局内状态只保留必要信息：

```text
{题阵类型} · 第 {n} / 5 题
目标：{本局目标}
流派：{当前流派}
破招：{当前破招}
```

不要在局内展示完整学习域报告、题库审计数字或长段说明。

## 5. 破招文案

破招文案要让玩家知道选择的学习含义。

| 破招 | 文案方向 |
|---|---|
| 稳破 | 正常作答，保持稳定收益 |
| 强攻 | 有把握时使用，答对收益更高 |
| 观照 | 先看一句提示，降低收益但降低盲猜 |

示例：

```text
观照：先看一句题眼提示。本题收益降低，但错因判断更清楚。
```

## 6. 答对反馈

答对反馈包含：

- 正确结果。
- 题眼短课。
- 本局目标变化。

模板：

```text
答对。

题眼短课：
{考点句}
{题眼句}

本局进度：{目标进度}
```

## 7. 答错反馈

答错反馈包含：

- 正确答案。
- 玩家答案。
- 题眼短课。
- 心魔变化。

模板：

```text
答错。
正确答案：{correct}
你的答案：{selected}

题眼短课：
{考点句}
{题眼句}
{错因句}

心魔变化：
{心魔类型} {变化}
```

## 8. 题眼短课

题眼短课是最重要的学习文字。

结构：

```text
考点：……
题眼：……
错因：……
```

写法要求：

- 不复述整段解析。
- 不泛泛写“请认真审题”。
- 必须点出本题的关键限定、主体、概念边界或选项陷阱。
- 默认收起完整解析，只显示短课。

## 9. 心魔文案

心魔文案要把失败翻译成可处理的问题。

示例：

```text
审题失误心魔增强。
这不是知识空白，而是漏看了题干里的限定词。
下一次先看否定词、主体和条件。
```

常见心魔解释：

| 心魔 | 解释方向 |
|---|---|
| 概念混淆 | 相近概念边界没分清 |
| 审题失误 | 否定词、限定词、主体或条件漏看 |
| 记忆盲区 | 基础事实或条文没有形成记忆 |
| 应用失误 | 知识点会背，但场景匹配错 |
| 多选漏选 | 只找到部分正确项，没有逐项排查 |
| 过度自信 | 信心高但作答错误，需要校准判断 |

## 10. 学习报告文案

学习报告文案必须回答：

- 本局做了什么。
- 目标是否完成。
- 主要错因是什么。
- 下一局建议是什么。

模板：

```text
本局报告

{题阵类型} · {流派}
正确率：{correct} / 5
本局目标：{goal}
完成情况：{progress}

主要问题：{primaryIssue}
下一局建议：{nextAction}
```

## 11. 禁止文案

- 大段世界观介绍。
- 与本局目标无关的闲聊。
- 只说“再接再厉”但不给原因。
- 只说“答错了”但不给错因。
- 让玩家先读长文才能作答。
- 用内部审计字段解释玩家行为。
