# 小明书院：秘卷巡游

一个无后端、无外部依赖的浏览器备考学习游戏。当前版本已经从“章节地图式 RPG”改造为“Roguelite 单局刷题”：玩家进入后先看到开局台，选择本局目标和学习流派，然后完成 5 题题阵，在战报里获得错因、心魔和下一局建议。

## 运行

```bash
cd /Users/wangzexu/Documents/Codex/academy
npm run start
```

浏览器打开：

```text
http://localhost:4190
```

如果 4190 被占用，可以用任意静态服务器托管当前目录，例如：

```bash
python3 -m http.server 4180
```

## 当前玩法

- 第一屏是开局台，不再是世界地图或章节路线。
- 系统根据当前薄弱点推荐一局：新题优先走探索局，有活跃心魔优先走净化局，复习期可走冲刺局。
- 每局开始前选择学习流派：稳修适合稳定推进，突击适合短时冲刺，复盘适合错题清理。
- 局内只显示本局目标、遭遇进度、题干、选项、破招式和必要反馈，不显示审计类题库信息。
- 剧情不再是独立阅读页，只作为开局、遭遇、结算里的事件化反馈出现。
- 战报会说明正确率、收获、产生或净化的心魔、主要错因，并直接给出下一局按钮。
- 心魔页用于查错题债；日课和报告页用于看长期学习状态。

## 题库格式

页面仍保留 JSON 导入入口。格式支持：

```json
{
  "questions": []
}
```

或直接使用题目数组：

```json
[]
```

单题字段：

```json
{
  "id": "question-001",
  "year": "2026",
  "type": "单项选择",
  "topic": "教育法规",
  "stem": "题干文本",
  "options": [
    { "key": "A", "text": "选项 A" },
    { "key": "B", "text": "选项 B" }
  ],
  "answer": "B",
  "explanation": "讲解文本",
  "sourceRef": "讲解来源",
  "difficulty": 2
}
```

多选题的 `answer` 写连续字母，例如 `ABC`。
